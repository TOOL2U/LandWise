# Image Assets Guide

Place your image files in this directory with the following names:

## Required Images

1. **01_hero.png**
   - Hero section background image
   - Recommended size: 1920x1080px or larger
   - Shows Ko Pha Ngan landscape/coastline

2. **03_package1_visual.png**
   - Land Snapshot package visualization
   - Recommended size: 600x400px
   - Shows drone map or 2D terrain view

3. **04_package2_visual.png**
   - Land Visibility Report package visualization
   - Recommended size: 600x400px
   - Shows PDF report mockup or visibility analysis

4. **05_package3_visual.png**
   - Land Ready Package visualization
   - Recommended size: 600x400px
   - Shows before/after land clearing or survey equipment

5. **06_deliverables_section.png** (Optional)
   - Reference image for deliverables section styling
   - Not displayed on site, used for design reference

6. **07_footer.png** (Optional)
   - Reference image for footer styling
   - Not displayed on site, used for design reference

## Image Guidelines

- Format: PNG or JPG
- Optimize images before uploading (use tools like TinyPNG)
- Ensure images are high quality but web-optimized
- All images will be automatically optimized by Next.js

## Temporary Placeholders

Until you add the actual images, the site will show:
- Gradient backgrounds
- Icon placeholders
- Or you can use the attached mockup image as reference
